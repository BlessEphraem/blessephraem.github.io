# ✦ EPHRAEM ✦

> **FROM ADOBE FRUSTRATION TO OPEN SOURCE SOLUTIONS.**

I didn't start coding just for the sake of it.
I started because I was tired of the limitations, bugs, and bloatware found in major creative suites (looking at you, **Adobe** 😒).
I realized that if I wanted tools that actually worked for me, I had to build them myself.

My philosophy:
- **🚫 Anti-Data Hoarding:** I dislike the fact that big tech owns our digital lives.
- **🔒 Local First:** Cloud is an *option*, not a requirement.

---

## 💻 Apps

- **[InputBar](./programs/inputbar/getting-started)** — Fast application launcher for Windows, triggerable with any shortcut (even the `Win` key).
- **[Premiere Companion](./programs/premiere-companion/getting-started)** — An Excalibur / BetterMost alternative that can also apply Presets. Free and Open Source.

## 🔌 Plugins

- **[JSX Runner](./plugins/jsx-runner)** — Premiere Pro UXP plugin to run `.jsx` scripts.
- **[JSX Scripts](./plugins/jsx-scripts)** — `.jsx` scripts free to use with JSX Runner.
